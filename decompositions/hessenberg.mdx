---
title: "Hessenberg Reduction"
description: "Reduce a matrix to upper Hessenberg form A = Q·H·Q† using Householder transformations."
---

Hessenberg reduction transforms a square matrix `A` into:
- **Q** — a unitary matrix
- **H** — an upper Hessenberg matrix (zero below the first subdiagonal)

such that `A = Q·H·Q†`.

An upper Hessenberg matrix has the form:

```
* * * * *
* * * * *
0 * * * *
0 0 * * *
0 0 0 * *
```

## Function

```go
func Hessenberg(a *matrix.Matrix, tol ...float64) (q *matrix.Matrix, h *matrix.Matrix)
```

Uses successive Householder reflections to zero out sub-subdiagonal entries column by column, updating both H and Q in place.

## Parameters

<ParamField path="a" type="*matrix.Matrix" required>
  The square input matrix to reduce.
</ParamField>

<ParamField path="tol" type="...float64">
  Optional tolerance overrides. Defaults: `AbsTol=1e-8`, `RelTol=1e-5`.
</ParamField>

## Return values

<ResponseField name="q" type="*matrix.Matrix">
  Unitary matrix Q such that A = Q·H·Q†.
</ResponseField>

<ResponseField name="h" type="*matrix.Matrix">
  Upper Hessenberg matrix H.
</ResponseField>

## Usage

```go
package main

import (
	"fmt"

	"github.com/itsubaki/decomp"
	"github.com/itsubaki/decomp/matrix"
)

func main() {
	a := matrix.New(
		[]complex128{1, 2, 3},
		[]complex128{2, 5, 6},
		[]complex128{3, 6, 9},
	)

	Q, H := decomp.Hessenberg(a)

	fmt.Println("Q unitary:", Q.IsUnitary())
	fmt.Println("H Hessenberg:", H.IsHessenberg())
	fmt.Println("Q*H*Q† == A:", matrix.MatMul(Q, H, Q.Dagger()).Equal(a))
}
```

## Complex Hermitian matrices

```go
a := matrix.New(
	[]complex128{2 + 0i, 1 - 1i, 0},
	[]complex128{1 + 1i, 3 + 0i, 4 - 2i},
	[]complex128{0, 4 + 2i, 1 + 0i},
)

Q, H := decomp.Hessenberg(a)
fmt.Println("Q*H*Q† == A:", matrix.MatMul(Q, H, Q.Dagger()).Equal(a))
```

## Why Hessenberg form matters

Hessenberg form is a preprocessing step before Schur decomposition. QR iteration on a Hessenberg matrix converges faster and is cheaper per iteration because each step preserves the Hessenberg structure (only O(n²) operations vs O(n³) for a full matrix).

In practice, for large matrices:
1. Reduce to Hessenberg with `Hessenberg`
2. Run QR iteration via `Schur` on H
3. Extract eigenvalues from the diagonal

<Tip>
If you only need eigenvalues and your matrix is already symmetric/Hermitian, `EigenJacobi` may be more convenient since it directly handles Hermitian inputs.
</Tip>
