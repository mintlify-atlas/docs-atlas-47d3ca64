## Installation

Add `decomp` to your Go module:

```bash
go get github.com/itsubaki/decomp
```

Requires Go 1.24 or later.

## Example 1: QR decomposition

Factor a matrix into Q (unitary) and R (upper triangular) such that A = Q·R.

```go
package main

import (
	"fmt"

	"github.com/itsubaki/decomp"
	"github.com/itsubaki/decomp/matrix"
)

func main() {
	a := matrix.New(
		[]complex128{1, 2},
		[]complex128{3, 4},
	)

	Q, R := decomp.QR(a)

	fmt.Println("Q unitary:", Q.IsUnitary())
	fmt.Println("R upper triangular:", R.IsUpperTriangular())
	fmt.Println("Q*R == A:", matrix.MatMul(Q, R).Equal(a))
}
```

To use Householder reflections instead of Modified Gram-Schmidt, swap `decomp.QR` for `decomp.QRHH` — the signature is identical.

## Example 2: Eigendecomposition and matrix powers

Compute the square root of a matrix using eigendecomposition and `Fdiag`.

```go
package main

import (
	"fmt"
	"math/cmplx"

	"github.com/itsubaki/decomp"
	"github.com/itsubaki/decomp/matrix"
)

func main() {
	x := matrix.New(
		[]complex128{0, 1},
		[]complex128{1, 0},
	)

	// EigenJacobi requires a Hermitian matrix
	V, D := decomp.EigenJacobi(x, 10)

	// Apply sqrt to the diagonal eigenvalues
	D.Fdiag(func(v complex128) complex128 { return cmplx.Pow(v, 0.5) })

	sqrtx := matrix.MatMul(V, D, V.Dagger())

	// Verify: sqrtx * sqrtx == x
	fmt.Println(matrix.MatMul(sqrtx, sqrtx).Equal(x)) // true
}
```

<Tip>
`EigenJacobi` requires the input to be Hermitian (A = A†). For general matrices, use `EigenQR` instead.
</Tip>

## Example 3: Matrix functions via Parlett

Compute a matrix function f(T) on an upper triangular matrix using Parlett's recurrence.

```go
package main

import (
	"fmt"
	"math/cmplx"

	"github.com/itsubaki/decomp"
	"github.com/itsubaki/decomp/matrix"
)

func main() {
	pow := func(p complex128) (decomp.ParlettF, decomp.ParlettF) {
		return func(z complex128) complex128 {
				return cmplx.Pow(z, p)
			}, func(z complex128) complex128 {
				return p * cmplx.Pow(z, p-1)
			}
	}

	t := matrix.New(
		[]complex128{2, 2, 3},
		[]complex128{0, 2, 5},
		[]complex128{0, 0, 3},
	)

	f, df := pow(2)
	t2 := decomp.Parlett(t, f, df)

	// t^2 via Parlett == t*t
	fmt.Println(t2.Equal(matrix.MatMul(t, t))) // true
}
```

<Note>
`Parlett` requires an upper triangular input. Use `Schur` first to reduce a general matrix to upper triangular form.
</Note>
