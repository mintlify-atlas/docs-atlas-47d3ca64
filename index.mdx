`decomp` is a Go library that provides a suite of numerical linear algebra decomposition algorithms. It operates on complex-valued matrices (`complex128`) and offers pluggable algorithm selection, configurable tolerances, and a clean, idiomatic Go API.

## Key design principles

- **Complex-valued throughout** — all matrices use `complex128`, supporting both real and complex problems
- **Pluggable QR backend** — select `QR` (Modified Gram-Schmidt) or `QRHH` (Householder) via the `QRFunc` interface
- **Configurable tolerances** — every algorithm accepts optional `tol ...float64` arguments for numerical precision
- **Zero dependencies** — pure Go standard library, no external packages required
